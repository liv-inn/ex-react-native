# Meu Perfil Profissional

App simples em React Native que exibe informações do perfil profissional.

## Como executar

1. Instale as dependências:
   ```bash
   npm install
   ```

2. Execute o app:
   ```bash
   npm start
   ```

## Funcionalidades

- Exibição de foto do perfil
- Dados pessoais
- Informações de formação
- Experiência profissional
- Lista de projetos
